What is Loan Approval Prediction?

This application uses machine learning to predict whether a loan application will be approved or rejected based on various applicant features such as income, credit history, education, and more.

How It Works

  • Data Collection: We gather loan application data including income, credit score, education, etc.
  • Data Preprocessing: Clean and encode categorical variables for machine learning
  • Model Training: Train multiple classification models (Decision Tree, SVM, Random Forest)
  • Validation: Test models on unseen data and select the best performer
  • Prediction: Use the trained model to predict loan approval for new applications

Machine Learning Models Used

  • Decision Trees: Makes decisions based on feature values in a tree structure
  • Support Vector Machines (SVM): Finds optimal decision boundaries between classes
  • Random Forest: Ensemble of decision trees for improved accuracy

Key Features Analyzed

  • Applicant and Co-applicant Income
  • Loan Amount and Term
  • Credit History (Most Important Factor)
  • Education Level
  • Employment Status
  • Property Area
  • Marital Status and Dependents
